# Fatal Encounter

Welcome to the Github page of Fatal Encounter

Fatal Encounter is a turn-based Battle game, where the player fights againts an enemy. 
Player can attack, heal or defend during their turn. The game ends when either the player or enemy's health reaches zero.

# About
Game started as a Terminal based game back near the end of 2024, where you had choise from 1-3. Eather attack, defend or heal. 
Later on, UI was added using tkinter, for easier game play, aditional shop and upgrades section, and patch notes in the game.

# How to play
You start at the main menu and select either Start game or Story Mode. When you go to start game, you are prompted to enter your name and the name of your enemy.
When the game starts, it starts a turn-based game. You can Attack, Defend or Heal (Damage and Shield are based in the upgrades shop).
